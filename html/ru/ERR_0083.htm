Сообщения и ошибки



Имя модели не может быть использовано

Эта ошибка возникает, когда вы пытаетесь использовать в SCM-файле без заголовка имя модели, значение ID которой не известно компилятору. Чаще всего это происходит, когда вы компилируете CLEO-скрипт с именем модели, которая в оригинальной main.scm содержится в списке DEFINE OBJECT (например, #KEYCARD). Такие #имена в отличие от моделей, которые прописаны в vehicles.ide, peds.ide default.ide, не могут быть скомпилированы.

Самое яркое проявление такой ошибки – в игре вместо ожидаемого вам объекта появляется иконка информации (#INFO).

Возможные решения: используйте числовой идентификатор модели (ID) вместо имени. Например, для #KEYCARD ID = 1581. Найти нужный ID можно в IDE-файлах игры. Также вы можете воспользоваться встроенной в SB функцией конвертации имени модели в ID и наоборот (Ctrl+Alt+H).
