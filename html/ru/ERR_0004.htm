Сообщения и ошибки

 

Неизвестный параметр

Каждый опкод имеет ограниченное количество типов параметров, каждый из которых обозначен определенным числом. Если данное число неизвестно декомпилятору, или оно выходит за границы типов параметров для данной версии игры, возникает эта ошибка. Ошибка возникает, если файл был скомпилирован неправильно, либо число параметров для декомпилируемого опкода не совпадает с оригинальным.

Возможные решения: включите через консоль пропуск неизвестных опкодов (команда toggle ignore_unknown).
