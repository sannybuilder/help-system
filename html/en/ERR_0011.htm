Sanny Builder Error Messages

 

Unknown opcode at

If the decompiler found an opcode, that the INI file does not contain, it shows this error message. In most cases, it happens when the number of parameter of the previous opcode does not match the INI number.

Possible solutions: either add the mentioned opcode in the proper INI file, or toggle debug option ignore_unknown via console.
