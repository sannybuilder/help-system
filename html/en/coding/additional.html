Built-in commands

The following commands are built in directly in the compiler code.

INC

Increments the first parameter by the second one. The first parameter is a variable.

Inc($IntVariable, $Value) => $IntVariable += $Value

Second parameter is equal to 1 if isn't specified.

Inc(1@) => 1@ += 1

Since version 3.06 it's possible to use a C++-like operator ++ that can be applied to variables and increments the value they hold by one.

$var++ => $var += 1

DEC

Subtracts the first parameter by the second one. The first parameter is a variable.

Dec($IntVariable, $Value) => $IntVariable -= $Value

Second parameter is equal to 1 if isn't specified.

Dec(1@) => 1@ -= 1

Since version 3.06 it's possible to use a C++-like operator -- that can be applied to variables and decrements the value they hold by one.

$var-- => $var -= 1

MUL

Multiplies the first parameter by the second one. The first parameter is a variable.

Mul($IntVariable, $Value) => $IntVariable = $IntVariable * $Value

Second parameter is equal to 2 if isn't specified.

Mul(1@) => 1@ = 1@ * 2

DIV

Divides the first parameter by the second one. The first parameter is a variable.

Div($IntVariable, $Value) => $IntVariable = $IntVariable / $Value

Second parameter is equal to 2 if isn't specified.

Div(1@) => 1@ = 1@ / 2

(internal) ALLOC*

Tells the compiler how to compile a global variable with a custom name (i.e. $text). For the variables with numeric names it has no effect.

Alloc($MyVar, 40) => the variable $MyVar will be compiled as $40

See also 'help\examples\alloc.txt'

SQR

Calculates the square of a variable passed as the parameter and stores the result in this variable.

sqr($var) => $var *= $var

Type of the variable has to be defined. See Variables for more information.

RANDOM

Generates a random number within a range specified by the parameters.

$rnd = random(1, $high)

This function can be used for both integers and floats. The opcode is chosen depending on the type of a resulting variable ($rnd in this case)
